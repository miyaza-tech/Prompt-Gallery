📝 Prompt Gallery 사용 가이드
기본 기능
1. 새 프롬프트 추가

상단의 "New Item" 버튼 클릭 -> 팝업 형태
이미지 추가 (선택사항):
URL 방식: 이미지 링크 직접 입력
File Upload: 로컬 이미지 파일 업로드 (최대 5MB)
카테고리 선택: Nano, GPT, Midjourney, Video, Photo, real_ch, Casual_ch, Anime_ch, real_bg, Casual_bg, Anime_bg
프롬프트 내용 입력 (최대 1000자)
_sref 입력
"Add Prompt" 버튼으로 저장

2. 프롬프트 관리

썸네일 호버: 마우스를 올리면 _sref, prompt, edit 버튼 표시
_sref: 코드를 클립보드에 복사
prompt: 프롬프트 텍스트를 클립보드에 복사
edit: 기존 프롬프트 수정

3. 필터링

상단 필터 바에서 카테고리별로 프롬프트 필터링
"All" 버튼으로 전체 보기

4. 데이터 관리

Export: JSON 파일로 백업
Import: 이전에 백업한 JSON 파일 불러오기

데이터 저장
모든 데이터는 브라우저의 localStorage에 자동 저장
서버 없이 완전히 클라이언트 측에서 작동
정기적으로 Export 기능으로 백업 권장

주요 특징
✅ 완전한 오프라인 작동
✅ 이미지 지원 (URL 또는 Base64)
✅ 실시간 문자 수 카운터
✅ 반응형 그리드 레이아웃 4열
✅ JSON 내보내기/가져오기

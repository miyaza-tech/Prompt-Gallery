📝 Prompt Gallery 사용 가이드

🌐 사이트: https://miyaza-tech.github.io/Prompt-Gallery/

===========================================
일반 사용자 (로그인 불필요)
===========================================

1. 프롬프트 보기
   - 갤러리 형태로 모든 프롬프트 확인
   - 4열 그리드 레이아웃
   - 각 카드에 카테고리 배지 표시

2. 필터링
   - 상단 필터 바에서 카테고리 선택
   - 여러 카테고리 동시 선택 가능 (OR 조건)
   - 선택된 카테고리는 검은색 배경으로 표시
   - "All" 버튼으로 전체 보기

3. 프롬프트 복사
   - 카드에 마우스 올리기 (호버)
   - "_sref" 버튼: 스타일 참조 코드 복사
   - "prompt" 버튼: 프롬프트 텍스트 복사
   - 복사 완료 시 알림 메시지 표시

===========================================
관리자 (로그인 필요)
===========================================

1. 로그인
   - 우측 상단 "Login" 버튼 클릭
   - 이메일과 비밀번호 입력
   - 로그인 성공 시 "Logout", "New Item", "Import", "Export" 버튼 표시

2. 새 프롬프트 추가
   - "New Item" 버튼 클릭 (중앙 팝업 모달 열림)
   - 이미지 추가 (선택사항):
     * URL: 외부 이미지 링크 직접 입력
     * File: 로컬 파일 업로드 (최대 10MB, Supabase Storage에 자동 업로드)
   - 카테고리 선택:
     * 버튼 형태로 여러 개 동시 선택 가능
     * 선택된 카테고리는 회색 배경으로 표시
     * 카테고리: Nano, GPT, Midjourney, Video, Photo, real_ch, real_bg, US_ch, US_bg, JP_ch, JP_bg, etc
   - 프롬프트 입력 (최대 1000자, 실시간 글자 수 표시)
   - _sref 입력 (스타일 참조 코드)
   - "Add Prompt" 버튼으로 저장

3. 프롬프트 수정
   - 카드에 마우스 올리기
   - "edit" 버튼 클릭 (로그인 시에만 표시)
   - 수정 모달에서 내용 변경
   - 이미지 변경 시 기존 이미지는 자동 삭제됨
   - "Update" 버튼으로 저장

4. 프롬프트 삭제
   - 수정 모달 하단의 "Delete" 버튼
   - 확인 메시지 표시
   - 삭제 시 Supabase Storage의 이미지도 함께 삭제됨

5. 데이터 백업
   - "Export" 버튼: JSON 파일로 다운로드
   - 파일명: prompt-gallery-backup-YYYY-MM-DD.json
   - "Import" 버튼: 백업 파일 불러오기

===========================================
데이터 저장 방식
===========================================

✅ Supabase PostgreSQL 데이터베이스
   - 프롬프트, 카테고리, _sref, 이미지 URL 저장
   - 실시간 동기화로 여러 PC에서 동시 작업 가능

✅ Supabase Storage
   - 이미지 파일 저장 (최대 10MB)
   - 무료 플랜: 1GB 스토리지 (약 1,000개 이미지)
   - 자동 URL 생성 및 데이터베이스 연동

✅ 보안
   - Row Level Security (RLS) 적용
   - 읽기: 누구나 가능
   - 쓰기/수정/삭제: 인증된 사용자만 가능

===========================================
주요 특징
===========================================

✅ 실시간 동기화: 여러 기기에서 동시 사용 가능
✅ 다중 카테고리: 한 프롬프트에 여러 카테고리 할당
✅ 다중 필터: 여러 카테고리 동시 선택으로 필터링
✅ 이미지 자동 관리: 삭제 시 Storage도 자동 정리
✅ 모달 UI: 팝업 형태의 Add/Edit 폼
✅ 반응형 디자인: 모바일, 태블릿, 데스크톱 지원
✅ 4열 그리드 레이아웃
✅ 카테고리 배지 표시
